Компакт-диск



На этом компакт-диске вы найдете:

Мультимедийный видеокурс в формате Flash 8/9.



Как работать с компакт-диском:

  • После вставки CD в устройство чтения компакт-дисков, домашняя страница с оглавлением диска должна загрузиться автоматически. Если опция автозапуска CD у Вас отключена, откройте вручную файл index.exe, либо index.swf (в последнем случае необходимо, чтобы на Вашем компьютере был установлен Flash-проигрыватель версии не ниже 8).

  • Если Вы выбрали файл index.exe, то подтвердите его запуск в диалоговых окнах-предупреждениях, которые, возможно возникнут (в зависимости от установок параметров безопасности Вашей ОС).

  • Для работы с Flash-видеокурсом запустите файл index.exe (или перейдите по данной ссылке из оглавления диска) Медиа-учебник не требует дополнительной установки, но, если Вы собираетесь просматривать его через браузер, необходимо установить соответствующий модуль ActiveX с сервера Adobe (используя Интернет-соединение).

  • При работе с автономным проигрывателем (файлом index.exe) никакой установки делать не требуется. Для навигации по видеолекциям используйте элементы управления в верхней части окна, а для управления просмотром - типовую панель воспроизведения медиа-проигрывателя в нижней части окна.

  • Если видео покажется Вам нечетким, попробуйте переключиться в режим масштаба 1:1 - командой View / 100%, либо в полноэкранный режим - командой View / Full Screen (или нажатием сочетания Ctrl-F).



Требования к конфигурации компьютера:

  • Компьютер, оснащенный CD-ROM или DVD дисководом, а также процессором Pentium(r) 600 МГц или более производительным

  • 256 Мб оперативной памяти

  • Монитор, поддерживающий разрешение не ниже 1024 x 768 (при худшем разрешении Flash-видео можно просматривать, но изображение будет нечетким)

  • Звуковая карта, снабженная колонками или наушниками

  • Операционная система Windows 2000 (или более новая).
