НЕПАРАМЕТРИЧЕСКАЯ СТАТИСТИКА И ПОДГОНКА РАСПРЕДЕЛЕНИЯ: Этот модуль содержит множество так называемых непараметрических критериев для сравнения групп. В сравнении с параметрическими критериями (t-критерий, дисперсионный анализ), непараметрические критерии основаны на меньших предположениях о сущности и распределении сравниваемых переменных. Например, имеются критерии для переменных, содержащих только информацию о рангах (для переменных, измеренных в порядковой шкале). Такие данные могут появиться при опросе респондентов в потребительском обзоре, когда производится упорядочение их предпочтений относительно нескольких конкурирующих марок мыла.

Если ваши данные содержат информацию о рангах, воспользуйтесь ранговым дисперсионным анализом Фридмана. Если ваши данные по своей природе категориальны, например, содержат коды, определяющие один из двух возможных ответов, который (повторно) выбрал респондент, воспользуйтесь Q критерием Кохрена. За подробностями обратитесь к руководству пользователя STATISTICA.

СТАТИСТИЧЕСКИЕ ГРАФИКИ: Статистические графики обладают различными средствами для построения гистограмм, линейных графиков, диаграмм рассеяния для нескольких переменных. Эти графики предлагают широкий выбор методов для визуализации различий между переменными.
