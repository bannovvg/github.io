Сравнение средних/дисперсий в двух группах

Выберите одну из следующих возможностей:

ОСНОВНЫЕ СТАТИСТИКИ И ТАБЛИЦЫ: Этот модуль вычисляет t-критерий для независимых выборок, позволяющий сравнивать средние двух выборок. Предполагается, что данные в двух группах (выборках) нормально распределены. Если значение t статистики является значимым, то можно сделать вывод о том, что средние в двух группах различны. Итоговая таблица результатов для t-критерия включает F-критерий для сравнения дисперсий в двух группах; если его значение является статистически значимым, то можно сделать вывод о том, что дисперсии в двух группах различны. Модуль Основные статистики также содержит калькулятор, который позволяет сравнить данное выборочное среднее с любым введенным значением.

НЕПАРАМЕТРИЧЕСКАЯ СТАТИСТИКА И ПОДГОНКА РАСПРЕДЕЛЕНИЯ: Этот модуль содержит различные так называемые непараметрические критерии для сравнения групп. Их нужно использовать в случае, если неизвестно распределение сравниваемых переменных, а также если измерения проведены в бедных шкалах: номинальных или порядковых. Например, данные могут в действительности состоять из рангов, а не из точных измерений.

СТАТИСТИЧЕСКИЕ ГРАФИКИ: Статистические графики обладают различными средствами для построения гистограмм, линейных графиков, диаграмм рассеяния по группам. Эти графики предлагают широкий выбор методов для визуализации различий между группами.
