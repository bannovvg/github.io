Статистический советник

Статистики для групп, выборок, переменных

Выберите одну из следующих возможностей:

ОСНОВНЫЕ СТАТИСТИКИ И ТАБЛИЦЫ: Этот модуль содержит процедуры группировки, которые позволяют вычислить средние (и стандартные отклонения) для наблюдений разбитых на группы значениями одной или более группирующей переменной (см. Внутригрупповые описательные статистики - группировка). Например, вы можете вычислить итоговую таблицу средних переменной Доход по Полу (мужчина, женщина), группируя Пол по Виду занятости, и Пол, Вид занятости по Происхождению. Этот модуль строит графики и вычисляет статистики наблюдений разбитых на группы с помощью какой-либо группирующей переменной.

СТАТИСТИЧЕСКИЕ ГРАФИКИ: Статистические графики предоставляют широкий выбор категоризованных графиков (например, категоризованные гистограммы, линейные графики).
