Статистический советник

Описательные статистики для времен выживания/отказов

Используйте АНАЛИЗ ВЫЖИВАЕМОСТИ. Этот модуль строит стандартные таблицы времен жизни, а также вычисляет множительные оценки Каплана-Мейера. Имеются также опции для подгонки распределений специальных видов (Вейбулла, Гомперца, линейной интенсивности) к наблюдаемым временам жизни/моментам отказов.

При проведении стандартного анализа таблиц с временами жизни, наблюдаемые данные разделяются на несколько выбираемых пользователем временных интервалов. После этого программа вычисляет для каждого интервала различные статистики, например, долю выживших до окончания соответствующего интервала, долю умерших на этом интервале, интенсивность смертности на этом интервале (вероятность смерти/отказа за единицу времени) и др. Кроме того, можно вычислить процентили кумулятивной функции выживания. Как и в других модулях системы STATISTICA, для просмотра результатов пользователю доступно большое количество специальных и легко настраиваемых графиков.

Метод множительных оценок Каплана-Мейера вычисляет статистики, подобные представленным в таблице времен жизни, однако, они их вычисление происходит в некотором смысле 'непрерывно', т.е. для каждого отдельного момента отказа в наборе данных. Поэтому результаты использования этого метода не подвержены влиянию (произвольного) выбора числа интервалов, используемых для анализа.
