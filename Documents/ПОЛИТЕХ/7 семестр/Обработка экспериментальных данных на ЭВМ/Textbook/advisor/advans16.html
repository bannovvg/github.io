Сравнение средних/дисперсий в нескольких группах

Выберите одну из следующих возможностей:

ДИСПЕРСИОННЫЙ АНАЛИЗ: Этот модуль выполняет так называемый дисперсионный анализ или анализ вариаций (ANOVA). ANOVA используется для одновременного сравнения средних в нескольких группах, при этом предполагается, что сравниваемые переменные нормально распределены внутри групп и имеют одну и ту же дисперсию. Иными словами, в дисперсионном анализе предполагается, что различие между группами проявляется только в средних, но не в дисперсиях. ANOVA/MANOVA также содержит несколько критериев для проверки однородности дисперсии в группах.

КОМПОНЕНТЫ ДИСПЕРСИИ И СМЕШАННАЯ МОДЕЛЬ ANOVA/ANCOVA: Этот модуль представляет полный набор методов для анализа планов со случайными эффектами, включая опцию оценивания компонент дисперсии для таких эффектов. Он также удобен для анализа смешанных моделей (включающих случайные и неслучайные факторы), больших моделей главных эффектов (например планов, содержащих 200 и более уровней каждого фактора), планов с большим числом факторов без взаимодействий высших порядков и анализа с использованием весов наблюдений. В системе STATISTICA имеется несколько модулей, позволяющих провести дисперсионный анализ факторных и специализированных планов. Обзор модулей системы STATISTICA, выполняющих дисперсионный анализ, и типов планов, для которых они предназначены, имеется в разделе Методы дисперсионного анализа.

ДИСКРИМИНАНТНЫЙ АНАЛИЗ: Методы этого модуля позволяют определить переменные (называемые дискриминирующими), наиболее отчетливо показывающие различие между средними заранее определенных групп, и провести классификацию наблюдаемых объектов в эти группы.

НЕПАРАМЕТРИЧЕСКАЯ СТАТИСТИКА И ПОДГОНКА РАСПРЕДЕЛЕНИЯ: Этот модуль содержит различные так называемые непараметрические критерии для сравнения групп. Их нужно использовать в случае, если неизвестно распределение сравниваемых переменных или если измерения проведены в бедных шкалах: номинальных или порядковых. Например, данные могут в действительности состоять из рангов, а не из точных измерений.

СТАТИСТИЧЕСКИЕ ГРАФИКИ: Статистические графики обладают различными средствами для построения гистограмм, линейных графиков, диаграмм рассеяния по группам. Этот модуль предоставляет наиболее полное визуальное сравнение групп.
