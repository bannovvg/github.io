Deployment method

The IBM Setup Wizard provides two deployment methods.

Enterprise deployment. This option creates a network release area and let you customize that release area. You can customize that release area using the Configure tool. A customized release area enables users to install pre-set configurations of IBM products from the release area.

Desktop Installation From CD Image. Select this option to install the product directly from the IBM installation CD-ROM onto your desktop. This is not recommended for setting up a ClearCase production installation.
