Окно проводника проекта

Окно проводника проекта Project (рис. 1.12) очень похоже на аналогичное окно проводника системы Windows и позволяет легко и быстро просматривать состав и свойства выбранного проекта, перемещаться между проектами, если их открыто сразу несколько, копировать необходимые объекты из окна одною проекта в другой, как это осуществляется в проводнике системы Windows.

Проводник проекта можно вызвать командой Project Explorer (Проводник проекта) меню View (Вид) или комбинацией клавиш <Ctrl>+<R>.

Рис. 1.12. Окно проводника проекта Visual Basic 6

Панель управления окна проводника проекта содержит три кнопки, которые имеют следующее назначение:

Кнопка

Назначение

Открывает окно редактора с кодом программы, выбранного в проводнике объекта

Открывает в конструкторе форм выбранный объект

Включает/выключает отображение папок


 

При нажатии правой кнопки мыши в окне проводника появляется контекстное меню, содержащее команды для добавления, сохранения, удаления форм, элементов управления и других объектов. В контекстном меню продублированы действия кнопок окна проводника.
